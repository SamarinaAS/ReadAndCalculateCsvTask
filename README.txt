Для компиляции и сборки основной программы необходимо, находясь в папке ReadAndCalculateCsvTask, в командной строке linux набрать следующие команды:
$mkdir build
$cd build 
$cmake ..
$make

В результате в папке build появится исполняемый файл main.
Чтобы его запустить, необходимо набрать:
$./main "/path/file.csv"
где "/path/file.csv" - путь или название csv-файла.

Для программы также были написаны тесты с помощью библиотеки gtest. Для их компиляции и сборки необходимо установить флаг BUILD_TESTS в TRUE в строке:
$cmake .. -DBUILD_TESTS=TRUE

Для запуска тестов необходимо набрать (находясь в папке build):
$./tests
